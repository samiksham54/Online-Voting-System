📌 Description:

This project is a Backend-based Online Voting System developed using Node.js and Express.js. The system provides REST APIs to manage voters and candidates through complete CRUD operations (Create, Read, Update, Delete).

MongoDB is used as the database for storing voter, candidate, and vote data. The backend is structured using a modular approach with separate models and routes for better maintainability and scalability.

The application was tested using Postman to ensure proper functionality of all API endpoints.
